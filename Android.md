# Android

## Context

### 什么是Context？

关于应用程序环境的全局信息的接口。 这是一个抽象类，它的实现是由Android系统提供。 它允许访问特定应用的资源和类，以及向上调用应用程序级的操作，如启动活动，广播和接收Intent等。
